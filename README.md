## Environments
- Preview: https://main--vitamix-imp-test-10--aemdemos.hlx.page/
- Live: https://main--vitamix-imp-test-10--aemdemos.hlx.live/

## Installation

```sh
npm i
```

## Linting

```sh
npm run lint
```

## Local development

1. Install the [AEM CLI](https://github.com/adobe/helix-cli): `npm install -g @adobe/aem-cli`
1. Start AEM Proxy: `aem up` (opens your browser at `http://localhost:3000/`)
1. Open the `vitamix-imp-test-10` directory in your favorite IDE and start coding :)
